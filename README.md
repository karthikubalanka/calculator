# 🧮 Basic Calculator

A simple and responsive **Basic Calculator** built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations through an interactive user interface and is suitable for beginners learning front-end web development.

---

## 📌 Features

* Clean and modern user interface
* Responsive design
* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Decimal number support
* Clear (`C`) button
* Error handling for invalid expressions
* CSS Grid layout for button alignment

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the calculator
* **CSS3** – Styling and responsive layout
* **JavaScript (ES6)** – Calculator logic and event handling
